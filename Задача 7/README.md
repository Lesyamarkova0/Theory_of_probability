## Теория по χ<sup>2</sup>-распределению

Область определения: x ∈ [0, +∞)

#### Определение
Распределение χ<sup>2</sup> (хи-квадрат) с k степенями свободы называется распределение суммы квадратов k независимых случайных величин, распределенных по стандартному нормальному закону, то есть: 

<pre><code> χ<sup>2</sup> = ∑<sub>i=1</sub><sup>k</sup> z<sub>i</sub><sup>2</sup>, </code></pre>

где Z<sub>i</sub> (i = 1, 2, ..., k) имеет нормальное распределение N(0; 1).

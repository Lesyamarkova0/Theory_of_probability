### Законы распределения дискретных случайных величин

## Задача на биномиальное распределение

При тестировании нового программного обеспечения проводится 7 независимых запусков. Вероятность падения программы при каждом запуске составляет 0,4. Случайная величина X — количество падений программы.

Необходимо составить закон распределения количества падений программы; найти математическое ожидание, дисперсию, среднее квадратическое отклонение и моду; отобразить многоугольник распределения.
